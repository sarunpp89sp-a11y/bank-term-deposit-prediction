# 🏦 Bank Term Deposit Prediction

## 📌 Project Overview

This project analyzes a bank marketing dataset to predict whether a customer will subscribe to a term deposit. The project includes data preprocessing, exploratory data analysis (EDA), data visualization, and machine learning using Logistic Regression.

---

## 🎯 Objectives

- Clean and preprocess banking customer data.
- Perform Exploratory Data Analysis (EDA).
- Visualize customer behavior and campaign performance.
- Build a Logistic Regression model.
- Predict customer subscription to term deposits.
- Generate actionable business insights.
